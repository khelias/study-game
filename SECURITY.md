# Security Policy

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities.

Report security concerns privately through the contact links on
https://khe.ee. Include the affected URL or repository, a short reproduction
path, and the impact you believe the issue has.

## Supported Scope

Only the current `main` branch and the live deployment are supported. This is a
personal portfolio project, so there is no formal SLA, but I treat reports that
could affect users, data, deployment credentials, or service availability as
high priority.
